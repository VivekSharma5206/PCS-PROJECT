# PCS-PROJECT
About End-to-end baseband digital communication system implemented in Python. Covers PCM sampling &amp; quantization, line codes (NRZ/RZ/Bipolar), raised cosine pulse shaping, AWGN channel, matched filter receiver, and BER vs SNR waterfall curve.
Features
Signal generation using configurable sinusoidal sources
Uniform sampling and quantization
Pulse Code Modulation (PCM)
Polar NRZ, Unipolar NRZ, and Bipolar AMI line coding
Rectangular and Raised Cosine pulse shaping
AWGN channel simulation
Matched filter receiver
Eye diagram analysis
Bit detection and BER computation
BER vs SNR (Waterfall Curve) generation
